Rice Leaf Disease Prediction

This project focuses on leveraging the power of Convolutional Neural Networks (CNNs) to predict and classify diseases in rice leaves. By analyzing images of rice leaves, we developed a model capable of identifying various types of diseases with a high level of accuracy.

Key Features

  • Designed and trained a CNN model to predict rice leaf diseases.
  • Performed classification to categorize the types of diseases based on leaf images.
  • Processed and enhanced input images using image preprocessing techniques.
  • Built a user-friendly interface for uploading images and viewing predictions.

Technical Skills

  • Python (TensorFlow, Keras)
  • Image processing (OpenCV)
  • HTML, CSS, and JavaScript for front-end development
